方法1：

git branch --contains commit</code></pre>

方法2：查找reflog


git reflog show --all | grep a871742</code></pre>

会有类似如下的结果：


a871742 refs/heads/completion@{0}: commit (amend): mpc-completion: total rewrite</code></pre>

completion就是所在的branch

注：


git reflog show</code></pre>

等价于


git log -g --abbrev-commit --pretty=oneline</code></pre>
